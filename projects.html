    Flight Delay Prediction with Machine Learning (Python)

    Our 3rd year ML (Machine Learning) group project, led by myself and my teammate, focuses on using machine learning algorithms to detect credit card fraud. By analyzing a comprehensive dataset with a significant class imbalance, we applied Logistic Regression and Random Forest algorithms to build a model capable of accurately identifying fraudulent transactions. We achieved high AUPRC scores of 0.89 for Logistic Regression and 0.95 for Random Forest, ensuring reliable fraud detection.
